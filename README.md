# Expense Tracker Application

A simple and responsive **Expense Tracker** built with **React.js**.  
Manage your **daily income and expenses** and track your balance in real-time.

---

## Features
- Add income and expense transactions
- Delete transactions
- Calculate total balance dynamically
- Summary of total income and total expense
- Responsive UI for desktop and mobile

---

## Tech Stack
- **React.js** (useState, props)
- **JavaScript**
- **HTML & CSS**
- **Vite** for project setup
